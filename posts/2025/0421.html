〇、写在开篇：当代码遇上哲学 ​

"每个程序员都是折翼的哲学家，一边写着最理性的代码，一边思考着最玄学的报错"

本教程将用3种部署方式、5大核心模块拆解、7个隐藏关卡，带你通关这个看似简单实则暗藏玄机的主页项目。请准备好你的终端、咖啡因补给包，以及面对「明明本地能跑」玄学问题的勇气。

二、核心API搭建：让主页拥有灵魂 ​

2.1 天气系统 - 高德地图API申请 ​

  1. 访问高德开放平台
  2. 注册账号 → 控制台 → 创建新应用 → Web服务
  3. 复制生成的Key，填入.env文件：
env
VITE_WEATHER_KEY=你的高德API密钥
1

技术冷知识：天气查询接口实际调用的是：

https://restapi.amap.com/v3/weather/weatherInfo?key=YOUR_KEY&city=310000

3.2 极客派 - Docker化部署 ​

dockerfile
# 高级技巧：多阶段构建优化镜像大小
FROM node:16-alpine AS builder
WORKDIR /app
COPY . .
RUN npm install -g pnpm
RUN pnpm install && pnpm build

FROM nginx:alpine
COPY --from=builder /app/dist /usr/share/nginx/html
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]

5.1 天气不显示怎么办？ ​

  1. 检查高德API的Web服务权限是否开启
  2. 通过curl命令验证接口：
bash
curl "https://restapi.amap.com/v3/weather/weatherInfo?key=YOUR_KEY&city=310000"

六、安全加固：给你的主页穿上盔甲 ​

6.1 防止CDN盗链 ​

在Nginx配置中添加：

nginx
location ~* \.(ttf|woff2)$ {
  valid_referers none blocked yourdomain.com;
  if ($invalid_referer) {
    return 403;
  }
}
